1.0 StructureDefinition-nswhealthitocmedicationadministration

NSW Health iTOC Medication Administration

A medication administration, extended to include an optional reference to a diluent medication.

Examples

Basic MedicationAdministration

1.0.1 Formal Views of Profile Content

This profile builds on AUBaseMedicationAdministration.
